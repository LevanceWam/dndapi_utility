# dndapi Utility Tool

## Installation

``` javascript

npm install dndapi_test --save

```

## Usage

### Require

``` javascript

const util = require('dndapi_test');

```

### .debug

``` javascript

util.debug('A meaningful title', obj);

```

### Expected output

``` javascript

====================================
A meaningful title
====================================
obj
Fri Aug 19 2016 21:00:29 GMT-0400 (EDT)

```
